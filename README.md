# TrevorAddons

TrevorAddons is a client-side Fabric mod for Hypixel SkyBlock Trevor hunting.
It highlights Trevor animals with ESP boxes and optional tracers, plus an in-game settings screen.

## Features

- ESP boxes for Trevor animals
- Optional tracer line to Trevor animals
- Custom tracer color
- Adjustable tracer thickness
- In-game config UI
- Debug command for targeted entity inspection

## Commands

- `/trevoraddons` or `/ta`: Open settings
- `/trevoraddons settings` or `/ta settings`: Open settings
- `/trevoraddons debug [range]` or `/ta debug [range]`: Show debug info for the looked-at entity

## Compatibility

- Minecraft `1.21.10`
- Fabric Loader `>=0.19.2`
- Fabric API `0.138.4+1.21.10`

## Build

```bash
./gradlew build
```

The built jar is generated in `build/libs/`.
